# Fractal Explorer

Интерактивная страница для исследования фракталов — выбираешь тип, крутишь параметры и смотришь, как фрактал строится в реальном времени. Никаких зависимостей: чистый HTML + Canvas + JavaScript, открывается прямо в браузере.

## Возможности

- **22 типа фракталов**, сгруппированных по категориям:
  - **Комплексная плоскость**: Мандельброт, Жюлиа, Burning Ship, Мультиброт (z^n), фрактал Ньютона
  - **IFS / самоподобные фигуры**: треугольник и ковёр Серпинского, папоротник Барнсли
  - **Кривые (L-системы)**: снежинка Коха, кривые Госпера, Гильберта, Пеано, дракон Хартера-Хейтуэя, кривая Минковского
  - **Деревья и рекурсия**: дерево Пифагора, H-дерево, множество Кантора, губка Менгера
  - **Процедурные**: броуновское дерево (DLA), плазма (diamond-square), фрактальное пламя, горная гряда (midpoint displacement)
- **Интерактивные параметры**: глубина/итерации, константа c для Жюлиа (Re/Im), угол излома для Коха, число хребтов для горной гряды и другие — в зависимости от типа
- **Скорость построения** — фрактал анимированно "вырастает" от простого к целевой сложности вместо мгновенной отрисовки (можно отключить)
- **Размерность** — для каждого фрактала показана точная формула (там, где применима) или эмпирическое значение
- **История и авторство** — кто открыл/придумал фрактал, почему так назвали, интересный факт
- Зум и панорамирование для фракталов на комплексной плоскости (колесо мыши / перетаскивание)
- Футуристичный дизайн: анимированный звёздный фон, glassmorphism-панель, неоновая подсветка

## Запуск

Просто открой `fractals.html` в браузере — никакой сборки или сервера не требуется.

```
git clone https://github.com/eremin-andrey/fractal-explorer.git
cd fractal-explorer
open fractals.html
```

## Файлы

- `fractals.html` — актуальная версия (футуристичный дизайн)
- `fractals-classic.html` — более ранняя минималистичная тёмная версия дизайна

## Технологии

Ванильный JavaScript, HTML5 Canvas, CSS. Без сборщиков, без внешних библиотек и шрифтов.
